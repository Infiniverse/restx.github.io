Improving a page

You want to improve the page , thanks in advance for your contribution!

Contributions are done via Github Pull Requests. If you have never done any pull requests don’t worry, we suggest you use prose.io using the Edit with prose.io button below, it’s very simple. If you are familiar with github pull requests, you can obviously do the PR with github, use the Edit on github button below.

Using prose.io

Edit with prose.io

Editing page with prose.io is as simple as 1 2 3:

  1. sign in with github account
  2. edit page
  3. submit changes

Using github

Edit on github

You can also edit the page using github interface. You will need to fork the repo, make your changes, and submit a pull request. Remember to use the gh-pages branch as base branch.
